Java-动态代理《一》

  • Java

定义

为某个对象提供一个代理，以控制对这个对象的访问。

Java 在 JDK 1.3版本中加入了代理功能、Java 的代理分为 静态代理 和 动态代理。

静态代理是指、在运行前就已经知道代理实现类的字节码文件、代理类和接口的关系。

动态代理是 程序在运行时 JVM 根据反射等机制动态生成代理类字节码文件、代理关系也是在运行时决定的。

平时进行Android 开发中 经常会用到ADB命令 特将常用的命令收集起来 以便日后查找

开启 adb 服务和停止

使用 start-server 检查 adb 服务器进程是否在运行，如果未运行则启动它。kill-server 停止adb 服务

安装和卸载程序

文件导入和导出

使用 pull 命令 将文件从 设备中导入到电脑 指定目录、使用 push 命令 将文件导入到设备指定目录

  • Java

反射是什么

先看看 Oracle 官方给出的解释

反射是Java编程语言的一个特色。它允许一个执行Java程序检查,内部程序的属性和操作。例如 它可能为一个Java类来获得其所有成员和显示他们的名字

能从内部检查一个Java类,但在其他编程语言根本就没有这个功能。例如,C或c++程序无法实现这个功能

对于一个标准的Javabean 构建工具可以通过反射获取Java的属性信息、因为他们是动态加载的

反射其实就是在运行时获取Java类中的信息、属性、构造方法、对象实例、方法等
